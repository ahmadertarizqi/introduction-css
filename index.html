Apa itu CSS..? ü§î

CSS singkatan dari Cascading Style Sheet, yakni dokumen yang berisi definisi style untuk sebuah dokumen HTML atau untuk mengatur tampilan dari dokumen HTML, meliputi layout dokumen, pewarnaan dan tampilan font dan teks dan lain sebagainya. CSS tidak pernah lepas dari HTML karna selalu berpasangan

Analogi-nya seperti ini !!

  • HTML sebagai struktur dari bangunan/kerangka
  • CSS sebagai tampilan luar dari bangunan/kerangka agar terlihat menarik
  • JAVASCRIPT sebagai perilaku/aksi dari bangunan/kerangka
↓

Menggunakan CSS ada 3 Cara

  • Inline CSS, digunakan untuk tag element HTML tertentu
  • Internal CSS, digunakan di dalam bagian Head HTML
  • External CSS, digunakan pada file terpisah tidak jadi satu dengan HTML

CSS Syntax

  • Selector menunjukan ke elemen HTML yang ingin di tambah/ubah CSS nya.
  • Blok deklarasi berisi satu atau lebih deklarasi yang dipisahkan oleh titik koma.
  • Setiap deklarasi menyertakan nama properti CSS dan nilai, dipisahkan oleh titik dua.

Element, ID, & Class Selectors di CSS

Di CSS setiap pemanggilan selector Element, ID, dan Class berbeda-beda. Berikut syaratnya :

  • Menggunakan selector "Element" dengan cara memanggil elementnya langsung, misal nama element "namaelement"
  • Menggunakan selector "ID" dengan cara di awali huruf hashtag atau pagar "#namaid"
  • Menggunakan selector "Class" dengan cara di awali titik ".namaclass"

Konsep Box Model, Apa itu? ü§î

Box model adalah sebuah representasi sebuah kotak di dalam element HTML. Kita bisa mengatur ukuran, posisi, warna, maupun background.

Dalam gambar tersebut, border garis berwarna merah adalah elemen HTML menggunakan konsep box model

Untuk menerapkan konsep box-model, biasanya menambahkan CSS untuk reset setiap elemen HTML, kenapa seperti itu?ü§î. Karna setiap browser mempunyai engine yg berbeda otomatis terkadang berbeda-beda hasilnya dan perlu ditambahkan vendor-prefix. Untuk box model tambahkan CSS berikut :

Komponen Box Model

Margin
Border
Padding
Content
  • Margin, Digunakan untuk mengatur jarak spasi element atau element parentnya.
  • Border, Garis suatu Element atau batas garis.
  • Padding, Jarak spasi antara border dan content.
  • Content, Isian dari element, bisa berupa text atau elemen lain, gambar, dan lain sebagainya

CSS Position üíª

CSS Position adalah properti CSS yg menentukan jenis posisi yg digunakan suatu elemen, meliputi (static, relative, fixed, absolute, atau sticky). Ada lima value untuk properti position :

  • Static (statis)
  • Relative (relatif)
  • Fixed (tetap)
  • Absolute (absolut)
  • Sticky (lengket atau presisi)
